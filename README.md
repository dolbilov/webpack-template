# Webpack template

## 🔍 **Description**

Template for projects that use SCSS and Webpack5.

## ⬇️ **Installation**

Install NPM dependencies:

```sh
npm install
```

## 🚀 **Usage**

<hr>

Start developing using Webpack dev server:

```sh
npm run dev
```

<hr>

Build project for production:

```sh
npm run build
```
